Simulation Projects

  • BLDC Motor Control: Developed a comprehensive Simulink model of a Brushless DC Motor integrated with a buck converter and three-phase inverter. Implemented PID control to regulate speed and torque under dynamic load conditions. Evaluated performance for different PWM strategies.
  • Adaptive Cruise Control (ACC): Simulated ACC system using PID control, tested under stop-and-go and cut-in scenarios. Tuned PID parameters to minimize overshoot and ensure safe following distance.
  • Automatic Emergency Braking (AEB): Calibrated threshold parameters for braking response during pedestrian detection using a simulation environment, enhancing vehicle reaction time.
  • Lane Keep Assist (LKA): Used CARLA simulator to design and evaluate LKA system. Configured camera and steering control to maintain lane centering during curves and lane-change maneuvers.
  • Digital Dashboard + 3D CAD: Designed a functional digital dashboard interface and created 3D models of vehicle components using INTO Design and SolidWorks.
