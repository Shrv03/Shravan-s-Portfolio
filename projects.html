Simulation & Technical Projects

BLDC Motor Controller Simulation
  • Developed a Simulink-based PID control system for Brushless DC motor using a buck converter and inverter setup.
  • Implemented closed-loop speed control and analyzed system response across varying load conditions.
Adaptive Cruise Control (ACC) using PID
  • Created a MATLAB/Simulink simulation of ACC to maintain headway using radar data and throttle/brake actuation logic.
  • Validated system for stop-and-go scenarios and optimized PID gains for minimum overshoot and quick response.
Autonomous Emergency Braking (AEB) Validation
  • Simulated pedestrian crossing and calibrated warning, braking, and emergency thresholds.
  • Validated braking response with dummy objects and time-to-collision metrics in Simulink.
Lane Keep Assist (LKA) using CARLA
  • Developed LKA logic in CARLA simulator using OpenCV and lateral offset feedback.
  • Analyzed lane departure triggers, road curvature effects, and integration with camera-LiDAR fusion.
3D Dashboard & HMI Design
  • Designed a custom 3D dashboard UI prototype for automotive HMI using INTO Design Studio.
  • Visualized speed, ADAS alerts, and battery health using UI principles aligned with ISO 26262.
